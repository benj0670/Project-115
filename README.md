# Project 115

Welcome to **Project 115** - your comprehensive resource hub for creating authentic Call of Duty Zombies prop replicas!

## About

Project 115 provides detailed guides and resources for making high-quality prop replicas from the Call of Duty Zombies universe. Whether you're interested in 3D printing, electronics integration, or complete prop builds, you'll find the information you need here.

## Features

- 📚 **Comprehensive Guides**: Step-by-step tutorials for all skill levels
- 🖨️ **3D Printing Resources**: Detailed printing guides and post-processing techniques
- ⚡ **Electronics Integration**: Wiring diagrams and tutorials for adding lights, sounds, and interactive features
- 🎮 **Game-Accurate Replicas**: Focus on authenticity and attention to detail

## Getting Started

Visit the [Project 115 website](https://project-115.github.io) to explore all available resources.

## Development

This is a static GitHub Pages site built with HTML, CSS, and JavaScript.
